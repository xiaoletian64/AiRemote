📋 使用说明

1. 把遥控器和 Windows 蓝牙配对（系统设置 → 蓝牙 → 添加设备）
2. 配对后遥控器自动连接，按键即可使用
3. 在「按键」页选语音键模式（Win+H 听写 / Ctrl / Win / 麦克风开关）
4. 要用语音转麦克风：安装 VB-Cable（设置页有链接）

🎤 语音键映射模式（Windows 适配）

🎙️ 语音转麦克风（VB-Cable）

安装 VB-Cable 虚拟声卡后，遥控器语音可作为 Windows 麦克风输入。
下载：VB-Audio Cables 官网
